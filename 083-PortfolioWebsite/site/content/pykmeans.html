pykmeans is the Python code used to run the experiments for my MSc by dissertation---a postgraduate research degree--in Computer Science, specifically Machine Learning.

The title of the thesis was An Extensive Empirical Comparison of K-means Initialization Algorithms, and the library uses scikit-learn and implements concurrency using futures.ProcessPoolExecutor.

It's all on github, under the MIT licence. In fact, the project deliverables included three libraries in total. See also:

  • pygendata for generating synthetic data sets
  • pycleandata for cleaning real-world data sets, typically from ... pandas

They all use NumPy and Pandas quite a bit, and I forget what else.

Well, I got my degree, and my research got published as "An Extensive Empirical Comparison of K-means Initialization Algorithms" in IEEE Access, so it must have done its job!
